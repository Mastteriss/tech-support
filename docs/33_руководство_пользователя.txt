Руководство пользователя
"Система технической поддержки"

1. ВВЕДЕНИЕ
Система предназначена для эффективного управления обращениями 
клиентов в службе технической поддержки.

2. ОСНОВНЫЕ ВОЗМОЖНОСТИ

2.1. Управление вопросами
- Создание новых вопросов с указанием приоритета
- Просмотр всех вопросов с фильтрацией по статусу
- Отслеживание статуса вопроса (new → in_progress → resolved → closed)

2.2. Управление консультантами
- Добавление консультантов с указанием специализации
- Назначение консультантов на вопросы
- Отслеживание нагрузки консультантов

2.3. Работа с ответами
- Добавление ответов на вопросы
- История всех ответов
- Отметка вопроса как решенного после получения ответа

2.4. Статистика
- Общее количество вопросов
- Распределение по статусам
- Количество консультантов и ответов

3. ПОРЯДОК РАБОТЫ
3.1. Для создания вопроса заполните форму и укажите приоритет
3.2. Для назначения консультанта выберите вопрос и консультанта
3.3. Для добавления ответа укажите ID консультанта и текст
3.4. Для завершения работы отметьте вопрос как решенный

4. ТРЕБОВАНИЯ К СИСТЕМЕ
- Python 3.8+
- Установленные зависимости из requirements.txt
- Доступ к порту 5000 для сервера
