Руководство оператора
"Система технической поддержки"

1. ОБЩИЕ СВЕДЕНИЯ
Система предназначена для автоматизации работы технической поддержки.

2. ЗАПУСК СИСТЕМЫ
2.1. Запустите серверную часть:
     python server/app.py
2.2. Запустите клиентскую часть:
     python client/client.py

3. РАБОТА С СИСТЕМОЙ
3.1. Вкладка "Вопросы"
  - список всех вопросов с фильтрацией по статусу
  - просмотр деталей вопроса
  - создание нового вопроса
  - назначение консультанта
  - добавление ответа
  - отметка вопроса как решенного

3.2. Вкладка "Консультанты"
  - список консультантов с их специализациями
  - добавление нового консультанта

3.3. Вкладка "Статистика"
  - общая статистика по вопросам
  - распределение по статусам
  - количество консультантов и ответов

4. ОСНОВНЫЕ ДЕЙСТВИЯ
4.1. Добавление вопроса:
  - нажмите "Новый вопрос"
  - заполните поля и нажмите "Создать"

4.2. Назначение консультанта:
  - выберите вопрос в списке
  - нажмите "Назначить консультанта"
  - выберите консультанта из списка

4.3. Добавление ответа:
  - выберите вопрос в списке
  - нажмите "Добавить ответ"
  - введите ID консультанта и текст ответа
