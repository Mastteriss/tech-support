Описание применения
"Система технической поддержки"

1. НАЗНАЧЕНИЕ
Программа предназначена для автоматизации процессов технической 
поддержки клиентов в организациях любого масштаба.

2. ОБЛАСТЬ ПРИМЕНЕНИЯ
- Службы технической поддержки
- IT-отделы организаций
- Сервисные центры
- Компании, предоставляющие услуги поддержки

3. РЕШАЕМЫЕ ЗАДАЧИ
3.1. Организация рабочего процесса:
- централизованное хранение всех обращений;
- распределение нагрузки между консультантами;
- контроль сроков обработки обращений.

3.2. Повышение качества обслуживания:
- быстрый доступ к истории обращений;
- отслеживание статуса каждого вопроса;
- статистика работы консультантов.

4. ПОТРЕБИТЕЛИ
- Консультанты технической поддержки
- Руководители служб поддержки
- Администраторы IT-систем

5. ПРЕИМУЩЕСТВА
- Простота использования
- Быстрое развертывание
- Открытый исходный код
- Возможность расширения функционала
- Кроссплатформенность
